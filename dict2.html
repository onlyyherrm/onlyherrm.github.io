Python def Keyword - Deep Explanation

1. What is the def Keyword?

The def keyword is used to define a function in Python. A function is a block of reusable code that performs a specific task when called.

2. Basic Syntax

def function_name(parameters):
    """Optional: Docstring (explains what the function does)"""
    # Function body (indented)
    return value  # Optional

3. Example Function

def greet():
    print("Hello, welcome to Python!")
greet()

4. Parameters and Arguments

def add(a, b):
    return a + b

result = add(10, 5)
print("Sum:", result)

5. Return Statement

def square(num):
    return num ** 2

result = square(4)
print("Square of 4 is:", result)

6. Default Arguments

def greet(name="User"):
    print("Hello,", name)

greet("Alice")
greet()

7. Variable-Length Arguments

*args Example

def add_all(*args):
    print("Sum:", sum(args))
add_all(1, 2, 3, 4, 5)

**kwargs Example

def show_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")
show_info(name="Alice", age=25)

8. Functions as Arguments

def square(x):
    return x ** 2

def apply_function(func, value):
    return func(value)

result = apply_function(square, 7)
print("Result:", result)

9. Lambda Functions

square = lambda x: x ** 2
print("Square of 5:", square(5))

10. Methods in Classes

class Person:
    def __init__(self, name):
        self.name = name
    def greet(self):
        print(f"Hello, my name is {self.name}.")
p = Person("Alice")
p.greet()

11. Recursive Functions

def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)
print("Factorial of 5:", factorial(5))

12. Nested Functions

def outer():
    print("Outer function")
    def inner():
        print("Inner function")
    inner()
outer()

13. Decorators

def decorator(func):
    def wrapper():
        print("Before function call")
        func()
        print("After function call")
    return wrapper

@decorator
def say_hello():
    print("Hello!")
say_hello()

14. Key Takeaways

  • Use def to define functions.
  • Functions can accept inputs (parameters) and return outputs (return).
  • Use *args for multiple positional arguments, **kwargs for multiple keyword arguments.
  • Functions can be nested, recursive, or passed as arguments.
  • Decorators add extra functionality to functions.
